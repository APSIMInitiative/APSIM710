Modify a Graph component

With a graph that you have dragged from the Graph toolbox you can edit the size of the graph, add a Title to a graph, add a Title and maximum and minimum values for the X and Y axis. Here is how to do it.

Edit the size of a graph

  1. Right mouse click on the graph and select "Edit". This gets you into "Edit Mode".

  2. Click on the graph again.

  3. Stretch the borders of the graph to whatever size you like.

  4. Right click on the background again and select "Exit edit mode".

 

 

Add a Title to the graph, add a Title and maximum and minimum values for the X and Y axis.

  1. Right mouse click on the graph and select "Chart Properties".

  2. In the property window that comes up, you can add a Title to the entire graph by typing something next to the Title property selected in the image.

    You can then set the Maximum, Minimum and Title properties for the Bottom (X), Left (Y), Right and Top Axes on the graph by clicking the "+" symbol next to "Axes" and repeating for which axis you which to edit, and then you can enter a value next to the appropriate property.

  3. To finish right mouse click on the graph again and select "Chart Properties".
