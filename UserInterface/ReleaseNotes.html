Release Notes for APSoil

Version 5.6

  • Fixed the water tab, Manage Crop right click menu to allow a new crop parameterisation to be added.
  • Apsoil now doesn't fully expand the soils tree at start up. It now only shows the states in the tree
  • The comment box is now read/write.
  • The predicted crop parameterisation for Black and Grey vertosols now appear correctly.

Version 5.5

  • Added GPS Latitude, Longitude (WGS84), Accuracy and APSoil Number under the General tab.
  • Added Mn(mg/kg) and Al(meq/100g) under the Soil Profile tab.
  • Added an option to get the latest soils file from the web. See button titled Get latest data
  • Added a new button Release Notes that displays these release notes.
  • Fixed defect where the Change PH units button wasn't converting the PH column.
  • Fixed defect where the Export to spreadsheet option was mangling the summer and winter Cona and U dates during the transfer to Excel.
  • Changed the way predicted LL numbers are calculated for Black and Grey Vertosols. It now uses the top 3 layers of LL15 values for the top 3 layers of predicted crop LL values. The other predicted crop LL layers remain as they were in the previous version of APSoil.
  • Fixed defect where if you open a .soils file without the password (as any non administrator would), and then open a new file you would be unable to import data as the import features are greyed out.
